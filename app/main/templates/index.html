HAZEN

Quality assurance platform for MRI test objects

HAZEN is a software framework for performing automated analysis of magnetic resonance imaging (MRI) quality assurance data. Hazen consists of multiple image processing Tasks which perform quantitative analysis of MRI phantom data.

Hazen currently supports the ACR Large MRI Phantom and the MagNET Test Objects collection of phantoms.

HAZEN Tasks

Hazen provides the following measurements of MRI test objects:

  • Signal-to-noise ratio (SNR)
  • Spatial resolution
  • Slice position
  • Slice width
  • Uniformity
  • Ghosting
